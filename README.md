# Mystic ToDo (MAUI)

Mystic ToDo is a cross-platform productivity application built using .NET MAUI. It provides a unified system for managing tasks, reminders, schedules, and calendar-based planning in a single modern interface.

---

## 📖 Description

Mystic ToDo helps users organize daily tasks, recurring schedules, and grouped activities. It includes calendar integration, task categorization, and theme support to improve productivity and user experience.

The application is built using the MVVM architecture and uses SQLite for local data storage.

---

## 🛠 Technologies Used

- **Language:** C#
- **Framework:** .NET MAUI
- **Architecture Pattern:** MVVM (CommunityToolkit.MVVM)
- **Database:** SQLite
- **IDE:** Visual Studio 2026
- **Integrations:** Google Calendar APIs

---

## ✨ Key Features

- Task creation, editing, and deletion
- Group-based task organization
- Calendar view for scheduled tasks
- Recurring tasks and reminders
- Google Calendar event sync (via Google Events group)
- Light and Dark theme support
- SQLite local database storage
- Attachment support for tasks

## 🚀 How to Run the Application

### Prerequisites

- Visual Studio 2022 or later
- .NET MAUI workload installed
- Android Emulator (optional for Android testing)

---

## 📥 Installation

1. Clone the repository
2. Open the solution file:
   Mystic_ToDo(MAUI).sln
3. Restore NuGet packages (Visual Studio will usually do this automatically)
4. Select target platform:
   Windows Machine OR
   Android Emulator
5. Build and Run the application (Press ▶ Start)
